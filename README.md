# Personalized Quiz Tool

This Python script demonstrates a personalized practice tool concept. It presents quiz questions and tracks performance by topic. Based on correct and incorrect answers, it dynamically adjusts "weakness scores" for each topic, prioritizing questions from areas where the user struggles most to provide a tailored learning experience.

## Language

`python`

## How to Run

Save the code as `main.py`. Open a terminal or command prompt, navigate to the directory where you saved the file, and run `python main.py`. Follow the prompts to answer questions.
